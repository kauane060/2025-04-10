Sobre Mim

Feira de Ciências 2023

Sobre o Projeto

Este projeto foi desenvolvido para a Feira de Ciências anual do Colégio Dom Pedro II em 2023, com o tema "Sustentabilidade e Reciclagem: Pequenas Ações, Grandes Impactos".

Em equipe, realizamos pesquisas sobre os hábitos de reciclagem na comunidade escolar, criamos maquetes interativas mostrando o ciclo de vida dos materiais recicláveis, e desenvolvemos um guia prático para separação correta do lixo.

Tecnologias Utilizadas

  • Pesquisa de campo e análise de dados
  • Materiais reciclados para as maquetes
  • Design gráfico para os materiais informativos
Coleção de Arte Digital

Sobre o Projeto

Minha coleção de arte digital começou como um hobby durante o período de aulas online, quando descobri ferramentas gratuitas de design e decidi explorar minha criatividade.

A coleção inclui mais de 30 peças criadas ao longo de um ano, explorando diferentes temas como natureza abstrata, retratos estilizados e ilustrações conceituais. Todas as obras foram criadas usando apenas um tablet básico e software gratuito.

Técnicas Utilizadas

  • Ilustração digital com camadas
  • Uso de pincéis personalizados
  • Exploração de paletas de cores
  • Composição e teoria da cor
Primeiro Site Pessoal

Sobre o Projeto

Meu primeiro site pessoal foi criado como um projeto de autoaprendizado em HTML e CSS básicos. A ideia era ter um espaço online para compartilhar meus interesses e projetos iniciais.

O site inclui seções sobre mim, meus hobbies, projetos escolares e objetivos futuros. Foi desenvolvido sem frameworks, usando apenas HTML e CSS puros, o que me permitiu entender os fundamentos do desenvolvimento web.

O Que Aprendi

  • Estrutura básica de HTML
  • Estilização com CSS
  • Design responsivo básico
  • Organização de projetos
  • Publicação de sites simples
